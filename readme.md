# Student Information System (Basic Version)

## Project Overview
This is a basic Django-based student information system developed for the XJCO3011 coursework. It includes user authentication, navigation pages, student record management, and JSON API endpoints.

## Features
- Homepage with system introduction
- User registration, login, and logout
- Student CRUD operations
- SQLite database integration
- Basic JSON API endpoints

## Technologies
- Python 3
- Django
- SQLite
